# Shopping Decision System

A web-based application that recommends products based on user preferences like budget and priority.

## Features
- Takes user input (product, budget, priority)
- Generates recommendations using decision logic
- Displays results with reason, tradeoffs, and confidence
- Simple web interface using Flask

## Tech Stack
- Python
- Flask
- HTML/CSS
- Pydantic

## How to Run

1. Clone the repository:
git clone <your-repo-link>

2. Navigate to project:
cd shopping-decision

3. Install dependencies:
pip install flask pydantic

4. Run the app:
python app.py

5. Open in browser:
http://127.0.0.1:5000

## Example Output
- Product: Laptop  
- Decision: Dell Inspiron 15  
- Reason: Good performance within budget  
- Tradeoffs: May compromise on other features  
- Confidence: Medium
